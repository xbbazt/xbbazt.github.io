重装系统脚本(仅适用于linux kvm)

出自: leitbogioro star 3k 项目地址

wget --no-check-certificate -qO InstallNET.sh 'https://raw.githubusercontent.com/leitbogioro/Tools/master/Linux_reinstall/InstallNET.sh' && chmod a+x InstallNET.sh

出自：MoeClub star 831 项目地址

bash <(wget --no-check-certificate -qO- 'https://raw.githubusercontent.com/MoeClub/Note/master/InstallNET.sh') -d 10 -v 64 -p "自定义root密码" -port "自定义ssh端口" -a -firmware

-firmware      额外的驱动支持 
-d             Debian系统 后面是系统版本号，例：9、10 ... 
-c             Centos系统 后面是系统版本号，例：6.9、6.10 ... 
-u             Ubuntu系统 后面是系统版本号，例：16.04、18.04 ... 
-v             系统位数，64位或32位，只写数字 
-a             auto，全自动无人值守安装 
-mirror       后面是指定镜像源地址 
-p             后面写自定义密码 
–ip-addr      ifconfig -a 后获取到的 例：194.87.xxx.xxx 
–ip-gate      route -n 后获取到的 例 194.87.xxx.xxx 
–ip-mask      255.255.xxx.xx

注意事项：

  • Vicer脚本目前不支持重装为CentOS 7系统，支持CentOS 6.9以下版本。

  • 重装的系统源自官方发行版。

  • 安装过程全自动进行，无需VNC操作，无需进入救援模式。

  • 系统安装完成后的默认用户名为：root，未自定义密码的话，默认密码为：MoeClub.org 。

  • 部分机器需要设置网卡，否则可以VNC，但是不能远程SSH。

  • 三毛、甲骨文 记得去掉 -firmware 参数，否则会出现错误。

  • 如果是腾讯云需要先卸载组件，否则会导致报错 DD失败。

vps综合性能测试脚本

融合怪 star 3.6k 项目地址

融合怪: 顾明思议就是把所有vps测试工具融合在一个脚本中了。内主要包含: 硬件与软件信息查询，IP和ASN查询，CPU评估，硬盘测速，流媒体解锁，IP质量检测，网络路由与速率等。包含了Github比较常用的开源项目!

使用方法:

curl -L https://gitlab.com/spiritysdx/za/-/raw/main/ecs.sh -o ecs.sh && chmod +x ecs.sh && bash ecs.sh

或

curl -L https://github.com/spiritLHLS/ecs/raw/main/ecs.sh -o ecs.sh && chmod +x ecs.sh && bash ecs.sh

YABS 项目地址

此脚本可自动执行业内最佳的基准测试工具。其中包括几项测试，用于检查服务器关键区域的性能：使用fio测试磁盘性能、使用iperf3测试网络性能以及使用Geekbench测试 CPU/内存性能。此脚本的设计无需安装任何外部依赖项，也无需提升权限即可运行。

使用方法:

curl -sL yabs.sh | bash

或

wget -qO- yabs.sh | bash
